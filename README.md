#  Stock Analyzer

A stock analysis tool built with **Excel + Python** that pulls market data from **Yahoo Finance** and displays key financial metrics, price charts, and news- all inside an Excel dashboard.


## What This Project Does

This tool lets you analyze stocks with **zero manual searching**.  

All you have to do is:

1. Enter a **ticker symbol**
2. Choose a **timeframe**
3. Click the button

The tool will:

- Fetch stock data from Yahoo Finance  
- Display important valuation + financial metrics  
- Generate a price chart based on your selected timeframe  
- Pull recent news articles for the stock  
- Compare up to **3 stocks** side-by-side with metrics and performance chart  


##  Features

### Single Stock Analyzer
- Enter ticker symbol (ex: `AAPL`, `TSLA`, `NVDA`)
- Select timeframe (1W, 1M, 6M, 1Y, etc.)
- Generates:
  - Stock price chart
  - Key metrics organized by financial statement category:
    - **Income Statement**
    - **Profitability & Margins**
    - **Growth**
    - **Valuation**
    - **Dividends**
    - **Risk**
  - Recent news and clickable links



### Recent News Feed
A dedicated **News sheet** shows:
- Article titles
- Publisher/source
- Date/time published
- Clickable “View Article” links


### 3-Stock Comparison Tool
Compare up to **3 tickers** at once:
- Side-by-side financial metrics
- Comparison chart showing % change over time
- Built-in visual table layout for easy interpretation
